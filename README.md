## 📌 Overview

CarrierNest is a **full-stack job portal application** that enables:

👨‍💼 Job seekers to explore and apply for jobs  
🏢 Recruiters to post and manage job listings  
🔍 Smart filtering for better job discovery  

This project demonstrates **real-world application architecture**, including authentication, data handling, and UI/UX design.

---

## ✨ Key Features

- 🔐 User Authentication (Login / Signup)
- 📄 Resume-based job applications  
- 🏢 Recruiter dashboard  
- 📊 Job posting & management  
- 🔍 Search & filter jobs  
- ⚡ Responsive UI
