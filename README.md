CarrierNest is a full-stack web application designed to bridge the gap between job seekers and employers. Built using Java J2EE technologies, it provides a seamless interface for job searching, application tracking, and recruitment management.

🚀 Key Features
User Authentication: Secure Login and Signup for both Job Seekers and Employers.

Job Management: Employers can post, update, and delete job listings.

Application System: Real-time job searching with filters and one-click applications.

MVC Architecture: Organized code structure for better scalability and maintenance.

Database Integration: Efficient data handling using SQL and JDBC.

🛠️ Tech Stack
Backend: Java (JSP, Servlets)

Database: MySQL

Connectivity: JDBC (Java Database Connectivity)

Frontend: HTML5, CSS3, JavaScript

Server: Apache Tomcat 9.0
